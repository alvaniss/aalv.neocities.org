half-life won
This page is made for a Half-Life WON config. This is a special version of the game made for speedrunning.

downloads:
goldsrc package
my config & soundpack
wireframe gun pack
livesplit
crosshair changer

instructions:
  • extract GoldSrc Package anywhere on your hard drive
  • go to: GoldSrc > Half-Life WON
  • extract downloaded file into the folder, replacing files
  • start the game by executing Half-Life (WON).bat
  • optional: install Wireframe Gun Pack, LiveSplit and Crosshair Changer
